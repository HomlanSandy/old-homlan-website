
注意事項:

1.確認『PHP的組態設定』與『更新包檔案大小』
  A. memory_limit設定，必須大於此更新包檔案檔案大小
  B. post_max_size設定，必須大於此更新包檔案檔案大小
  C. upload_max_filesize設定，必須大於此更新包檔案檔案大小
(欲了解或更改PHP的組態設定，請洽系統服務商)

2.後台線上更新限v2.0.1.0111101版本後可使用。
(v2.0.1.0111101以前版本，欲做更新請洽系統服務商)

3.請確認更新包皆為ZIP檔案格式，
檔名範例為『 ezportal_big5_v2.0.1.0111101_upgrade.zip 』。

4.至EZPORTAL網站後台，左方選單->參數設定->線上更新，按功能步驟說明執行即可。
